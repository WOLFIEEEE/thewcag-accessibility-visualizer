# TheWCAG Accessibility Visualizer Browser Extension

This directory contains the browser extension code for **TheWCAG Accessibility Visualizer**.

## About

**TheWCAG Accessibility Visualizer** is a browser extension that visualizes important but visually invisible accessibility information on web pages to help developers improve web accessibility.

This project is built on top of the original [Accessibility Visualizer](https://github.com/ymrl/a11y-visualizer) by [ymrl](https://github.com/ymrl), which provided the foundation and core functionality.

## Development

This extension is built with [WXT](https://wxt.dev/) and React. See the main [README.md](../../README.md) for development instructions.
